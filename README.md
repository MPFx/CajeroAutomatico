# Cajero Automático

Sistema de cajero automático desarrollado en Java.

## Documentación

Puedes ver el Javadoc aquí:  
[https://mpfx.github.io/CajeroAutomatico/](https://mpfx.github.io/CajeroAutomatico/)

## Funcionalidades

- Autenticación de usuarios
- Consulta de saldo
- Retiros y depósitos
- Transferencias entre cuentas
- Historial de transacciones
- Cambio de contraseña

## Tecnologías

- Java
- Javadoc
- GitHub Pages
